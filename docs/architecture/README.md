# Architecture Documentation (C4)

This folder contains the C4 architecture documentation for LTI ATS in Mermaid syntax.

- `c4-context.md`: level 1 context view (actors + external systems)
- `c4-containers.md`: level 2 container view (apps, services, data stores, queues)
- `c4-deployment.md`: deployment view mapped to Azure infrastructure
- `c4-dynamic-matching-flow.md`: dynamic sequence for AI matching execution

Recommended reading order:

1. `c4-context.md`
2. `c4-containers.md`
3. `c4-deployment.md`
4. `c4-dynamic-matching-flow.md`
